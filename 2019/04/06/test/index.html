前言

而后 Google 上查了下，发现国内有一个代码托管平台叫 Coding ，也可以将个人的 Hexo 博客托管到平台上，然后有点激动，说时迟那时快，立刻动手尝试将自己的个人博客托管到 Coding 上，中间遇到很多坑，折腾好久，最终成功，特此记录。 据了解，之前很多人都是把博客托管到国内的 gitcafe上，后来 gitcafe 被 Coding 收购。当然，据我所知国内还有其他代码托管平台，例如有个代码托管平台叫码云，但是码云能不能托管个人博客代码，我就不清楚了，没怎么用码云，基本用的都是 GitHub ，有需要的可以自行了解下。
